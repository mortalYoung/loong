# 第五章：龙隐地宫 — 皇权之谜

## 场景背景

五把钥匙齐聚，开启沉睡数十年的地下宫殿。融合机关、密道与阵法，层层深入方可抵达核心。龙隐秘藏的最终所在。

## 主线剧情

### 探索区域
- 龙纹长廊
- 九宫机关阵（天干地支解谜）
- 皇室密室（发现遗诏真相）
- 龙隐主殿

### 关键事件
1. 五钥开门，进入地宫
2. 龙纹长廊 — 龙纹指引方向
3. 九宫机关阵 — 六十甲子顺序破解
4. 皇室密室 — 发现遗诏：太祖对皇位传承另有安排
5. 守护者现身 — 白发老者出现，提出试炼
6. 最终之战

### Boss: 龙隐守护者
- 洪武年间秘密培养的守护者后人
- 三阶段战斗：剑术→机关陷阱→龙纹机关全面激活
- 目标：不让任何势力得到秘藏

## 最终抉择（四种结局）

击败守护者后，玩家面对遗诏做出选择：

1. **效忠朝廷** — 将秘藏交给建文帝，协助削藩
2. **扶助藩王** — 将秘密交予燕王势力，改变历史走向
3. **焚毁秘藏** — 彻底摧毁遗诏与机密，终结争夺
4. **隐世守护（隐藏结局）** — 联合各方封存秘藏，使其永不现世

## 本章揭示的真相

- 遗诏内容：太祖对皇位传承的真实安排

- 飞禽符号组织的最终身份
