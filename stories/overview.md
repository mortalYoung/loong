# 龙隐山河 — 故事梗概

## 时代背景

明朝建文元年，太祖朱元璋驾崩不久，新帝朱允炆推行削藩。各地暗流涌动，天下风雨欲来。

## 核心设定

洪武年间，朱元璋命人秘密修建「龙隐秘藏」——内藏山川布防图、军备密道图、足以左右皇权的遗诏。五把钥匙分藏五处，由不同势力暗中守护。

## 导火索

建文二年，奉旨调查队离奇失踪，仅一名重伤驿卒返京，临终留下四字："龙脉已开。"江湖中流出半张残缺地图，引来各方争夺。

## 四方势力

- **朝廷锦衣卫** — 巩固建文帝权位
- **燕王旧部** — 借秘藏改变天下局势
- **地方豪强/绿林** — 觊觎财富
- **隐世门派** — 守护秘密不现世

## 玩家身份

边境县城年轻游侠，为寻找失踪师父，偶得龙纹铜印，从此卷入天下大局。

## 五章旅程

1. **青石镇** — 调查师父失踪，收集线索，击败黑衣追兵
2. **鬼风岭** — 探索荒废驿站，发现调查队遗迹，获取第一把秘钥
3. **云隐寺** — 揭示秘藏真相，获取第二把秘钥
4. **黑石矿洞** — 三方势力周旋，获取第三、四把秘钥
5. **龙隐地宫** — 五钥开门，面对守护者，做出最终抉择

## 四种结局

1. **效忠朝廷** — 将秘藏交给建文帝，协助削藩
2. **扶助藩王** — 交予燕王势力，改变战争走向
3. **焚毁秘藏** — 让秘密永远埋葬
4. **隐世守护（隐藏）** — 联合各方封存秘藏

## 核心主题

每一次选择都影响人物命运，最终决定大明未来走向。
