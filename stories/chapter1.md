# 第一章：青石镇 — 风雨将至

## 场景背景

青石镇位于明朝西南边境，连接官道与山区的重要驿镇。商旅云集，却因近数月不断有人失踪而人心惶惶。玩家是镇上的年轻游侠，师父数月前外出未归。

## 主线剧情（6 段连锁事件）

### 线索收集

1. **师父住所调查** — 发现龙纹铜印和信件线索
2. **客栈问询** — 掌柜透露失踪事件，获得留言
3. **调查衙门** — 发现鬼风岭被官方封禁
4. **铁匠铺大火** — 铁匠为神秘人造武器后被灭口
5. **雨天冲刷** — 发现师父埋藏的手札："切勿相信县衙"

### Boss 战

6. **镇外截杀** — 集齐 5 条线索后，黑衣人拦路，触发 Boss 战

### Boss: 黑衣追兵
- 身手矫健的神秘黑衣人，奉命追杀持有龙纹铜印之人
- 击败后掉落：残缺地图①、解锁鬼风岭

## 支线剧情：失踪的小铁匠（5 段）

### 背景
铁匠老刘的儿子阿川突然失踪。阿川锻造技术出众，被黑衣人秘密带到山中据点打造龙纹兵器。

### 事件链
1. **铁匠老刘的恳求** — 老刘求助，透露龙纹模具失踪
2. **阿川的暗记** — 后山溪边发现阿川留下的划痕标记
3. **黑衣人的营地** — 潜入发现营地，龙纹兵刃数量远超山贼所需
4. **阿川的密信** — 木桩下找到密信，提到"燕王""靖难"
5. **夜救阿川** — 子时潜入救人，与黑衣头领交手

### 剧情意义
- 揭示黑衣人真实身份（燕王势力）
- 暗示即将到来的靖难之变
- 锻造龙纹兵器呼应主线铜印
- 为后续三方势力冲突埋伏笔

## 五条线索

| # | 名称 | 描述 |
|---|------|------|
| 1 | 龙纹铜印 | 师父留下的凭证 |
| 2 | 客栈留言 | 近期多人失踪 |
| 3 | 衙门公告 | 禁止进入鬼风岭 |
| 4 | 铁匠证词 | 神秘人订武器后灭口 |
| 5 | 师父手札 | "切勿相信县衙" |

## 本章揭示的真相

- 师父曾调查一张残缺地图，前往鬼风岭
- 有组织的黑衣人在追踪持有铜印之人
- 官府在掩盖失踪事件
- 黑衣人与某股势力有关（燕王旧部）
