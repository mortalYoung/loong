# 第四章：黑石矿洞 — 地下迷城

## 场景背景

洪武年间皇家矿场，突然封矿后地下被改造成巨大机关城。三方势力（锦衣卫、燕王旧部、江湖豪强）同时进入矿洞，各怀目的。

## 主线剧情

### 核心冲突
玩家需要在三方势力之间周旋：合作、欺骗或战斗。不同选择影响后续结局走向。

### 关键事件
1. 矿洞三岔口 — 发现三方势力都已到达
2. 锦衣卫的交涉 — 达成临时合作
3. 燕王暗卫 — 获知寨主弱点和幕后阴谋
4. 幕后真相 — 发现有人故意散播藏宝传言挑拨各方
5. 与血狼寨寨主对决

### Boss: 血狼寨寨主
- 高爆发，擅长召唤山贼协同作战
- 击败后获得第三、第四把秘钥

## 本章揭示的真相

- 幕后有人故意传播藏宝传说，目的是让各方互相争斗
- 飞禽符号组织的更大图谋
- 三方势力的真实目的
- 第三、四把秘钥
