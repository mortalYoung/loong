# 章节过渡叙事

## 设计原则

- **线索预埋**：本章暗示下一章的存在
- **Boss 掉落 = 通行证**：击败 Boss 获得前往下一章的物理凭证
- **动机升级**：个人（找师父）→ 天下（决定命运）
- **情感递进**：安全→恐惧→神秘→混乱→庄严
- **反向暗示**："切勿前往"反而强化前往的必要性

## Ch1 → Ch2（青石镇 → 鬼风岭）

**动机**：师父失踪，所有线索指向鬼风岭
**Boss掉落**：残缺地图 = 提供具体路线（知道去哪但找不到路）
**情感**：从"调查"进入"冒险"，离开安全镇子踏入未知

## Ch2 → Ch3（鬼风岭 → 云隐寺）

**动机**：遗书警告"切勿前往云隐寺" = 那里有答案
**Boss掉落**：铜虎符（第一把秘钥），背面刻"云隐"二字
**情感**：从"恐惧废墟"进入"神秘禅境"

## Ch3 → Ch4（云隐寺 → 黑石矿洞）

**动机**：老僧揭示五钥机制，卷轴指明第三四钥在矿洞
**Boss掉落**：秘藏卷轴 = 情报文件 + 三方势力预警
**情感**：从"单人探秘"进入"多方博弈"，格局升级

## Ch4 → Ch5（黑石矿洞 → 龙隐地宫）

**动机**：五钥已齐，各方势力逼近，必须抢先
**Boss掉落**：双龙钥匙（第三四把秘钥），五钥齐聚
**情感**：从"混乱争夺"进入"庄严终局"
